## ðŸ”§ Troubleshooting

### Masalah Umum

#### 1. GUI tidak terbuka
**Solusi**: Pastikan tkinter terinstall
```bash
python -m tkinter
```

#### 2. Error "Module not found"
**Solusi**: Pastikan semua file ada di direktori yang sama
```bash
ls -la *.py
```

#### 3. Input tidak valid
**Solusi**: 
- Biner: hanya 0 dan 1
- Desimal: hanya 0-9
- Oktal: hanya 0-7
- Heksadesimal: 0-9 dan A-F (huruf kapital)
